# (vswhere.exe). That is why you saw: "Could not find ... vswhere.exe".
#
# Fix (pick one):
#   A) Recreate .venv with GIL Python (NOT 3.13t). From project root:
#        .\setup_venv.ps1
#        .\.venv\Scripts\python.exe -m pip install -r requirements-ui.txt
#      Or manually: "C:\Program Files\Python313\python.exe" -m venv .venv
#   B) Use Python 3.12: py -3.12 -m venv .venv
#   C) Install "Build Tools for Visual Studio" with C++ (only if you must compile pandas).
#
# --- Install ---
#   .\.venv\Scripts\python.exe -m pip install -r requirements-ui.txt
#   .\.venv\Scripts\python.exe -m streamlit run dashboard.py
